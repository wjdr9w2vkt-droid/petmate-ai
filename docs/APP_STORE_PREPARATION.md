## 二、应用描述

### 短描述（170字符）

```
🐾 PetMate — 温暖的宠物陪伴空间。
AI助手解答养宠问题，时间轴记录成长每一步，
疫苗提醒不再忘记，纪念相册珍藏每一刻。
```

### 长描述

```
PetMate 不只是宠物管理工具。

它是一个温暖的宠物陪伴空间——
记录成长、保存回忆、守护健康。

✨ 核心功能

📖 成长时间轴
从到家第一天开始，记录疫苗、驱虫、体重、
美容、训练…每一个重要时刻。自动按年月排列，
滑动浏览，就像翻看一本成长日记。

📸 纪念相册
成长记录、时间轴事件中的所有照片自动聚合。
瀑布流、时间轴、年度回顾——三种方式，N种感动。

💉 疫苗 & 提醒中心
疫苗、驱虫、体检、生日、纪念日…
不再靠脑子记，PetMate 帮你想起来。
一键导出到 Apple Calendar。

🤖 AI 宠物助手
"狗狗不爱吃饭怎么办？" "猫咪掉毛正常吗？"
AI 宠物专家随时在线，温柔解答每一个疑惑。

📊 健康追踪
记录体重变化，自动生成趋势图。
周、月、年——看到成长的轨迹。

🔒 隐私安全
你的数据只有你能看到。
我们使用行业标准的行级安全加密。

🐶 为每一只宠物定制
支持多只宠物，每只拥有独立的档案、
时间轴、相册和提醒。

---

PetMate 是开源的，永远不会有广告。
我们相信，记录宠物的成长，
本身就是一件足够美好的事。
```

