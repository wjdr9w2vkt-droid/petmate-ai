# PetMate TestFlight 测试计划

---

## 一、测试目标

```
✅ 核心功能在真实 iPhone 上的表现
✅ PWA 安装体验
✅ AI 对话响应速度
✅ 照片上传/浏览体验
✅ 提醒创建/ICS 导出
✅ 多设备数据同步
✅ 发现 UI/UX 细节问题
```

## 二、测试员招募

### 理想测试员画像

```
5-15 人，包含:
  🐶 养狗用户 ×3
  🐱 养猫用户 ×3
  🐾 多宠用户 ×2
  🌱 新手宠主 ×2
  📱 iPhone 用户为主
```

### 招募渠道

```
朋友圈 / 宠物群 / 小红书
文案: "做了一个宠物App，求铲屎官帮忙测试 🙏"
```

## 三、测试任务

### 任务 1: 注册与入门（5分钟）

```
□ 打开 PetMate → 注册账号 → 验证邮箱
□ 观察: 注册流程顺畅吗？有困惑吗？
□ 设置昵称 + 头像
```

### 任务 2: 添加宠物（5分钟）

```
□ 新增一只宠物 → 填写档案 → 上传头像
□ 观察: 表单简单吗？有遗漏的字段吗？
□ 查看宠物详情页 → 浏览各 Tab
```

### 任务 3: 记录成长（5分钟）

```
□ 添加 3 条成长记录（不同日期、不同体重）
□ 查看体重趋势图
□ 切换 周/月/年
□ 观察: 趋势图容易理解吗？
```

### 任务 4: AI 对话（5分钟）

```
□ 配置 API Key（在个人中心）
□ 问 AI 3 个宠物问题
□ 观察: 回答有帮助吗？速度快吗？
```

### 任务 5: 时间轴（5分钟）

```
□ 添加几个时间轴事件（到家、疫苗、生日等）
□ 滑动浏览时间轴
□ 尝试编辑和删除
□ 观察: 交互直观吗？
```

### 任务 6: 提醒中心（5分钟）

```
□ 创建 3 条提醒（不同类型）
□ 尝试导出 ICS → 打开日历
□ 标记一条为完成
□ 观察: 提醒功能实用吗？
```

### 任务 7: 照片功能（5分钟）

```
□ 在记录/时间轴中添加照片
□ 打开纪念相册
□ 切换三种视图模式
□ 点击照片全屏浏览
□ 观察: 照片加载快吗？浏览体验好吗？
```

## 五、留存观察

### 7天观察指标

```
Day 1: 注册 + 添加宠物 + 首次记录 → 成功率？
Day 3: 是否再次打开？使用了什么功能？
Day 7: 是否还在使用？添加了多少条记录？
```

### 核心留存问题

```

❓ AI助手是否被重复使用？
```
