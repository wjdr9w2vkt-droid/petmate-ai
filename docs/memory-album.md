# 宠物纪念相册 — 功能设计方案

> 不是图库。是回忆空间。每一张照片都是爱过的证明。

---

## 一、功能定位

```
当前:  照片散落在各条成长记录中
目标:  统一照片浏览 + 瀑布流 + 纪念合集
```

---

## 二、照片来源

### 自动聚合

```
来源 1: growth_records.photo_url
来源 2: timeline_events.photo_urls
来源 3: pets.avatar_url（仅限历史头像）
来源 4: 独立上传（新增）
```

### 数据视图

```sql
CREATE VIEW pet_photos AS
SELECT
  id, pet_id, photo_url, recorded_at AS photo_date,
  'record' AS source
FROM growth_records WHERE photo_url IS NOT NULL
UNION ALL
SELECT
  id, pet_id, photo_url, event_date,
  'timeline' AS source
FROM timeline_events, unnest(photo_urls) AS photo_url;
```

## 四、交互

```
- 点击照片 → 全屏浏览（swipe 左右切换）
- 长按照片 → 分享 / 删除 / 设为封面
- 下拉刷新
- 滚动加载更多
- 全屏模式：黑色背景 + 日期水印 + 描述文字
```

---

## 五、纪念日照片

```
特殊日期自动高亮：

🎂 宠物生日 → 自动生成"生日照片集"
🏠 到家纪念日 → "一年了"回顾
🌟 用户自定义纪念日 → 专属相册
```

---

## 六、路由

```
/pets/[id]/photos          → 瀑布流相册
/pets/[id]/photos/timeline → 时间轴相册
/pets/[id]/photos/year     → 年度回顾
```
